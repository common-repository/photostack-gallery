PhotoStack allows you to create fullpage ajaxified image gallery.

== Description ==
This plguin is build to show photos from a gallery on any page. It is neeat way to show image gallery or work portfolio. This Plugin is inspired by blog on  http://tympanus.net/codrops/2010/06/27/beautiful-photo-stack-gallery-with-jquery-and-css3/ and used css and javascript code from its demo files. Check [live demo here](http://demo.amiworks.co.in/demo "WPoets Plyaground.").

This plugin is brought to you by [WPoets](http://wpoets.com/ "WordPress based development shop") development team.

== Installation ==


1. Upload `photostack-gallary` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now whenever you want to show gallery on any page add [show_gallery]
4. More setting will come soon.
